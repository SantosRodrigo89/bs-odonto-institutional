# BS Odonto - Site Institucional

Site one page para a clínica odontológica BS Odonto, especializada em DTM, bruxismo, ronco/apneia e odontologia estética.

## 🚀 Tecnologias

- **Next.js 14+** com App Router
- **React 18** com Hooks
- **Tailwind CSS** para estilização
- **TypeScript** para type safety
- **next/image** para otimização de imagens

## 📋 Pré-requisitos

- Node.js 18+ instalado
- npm ou yarn

## 🛠️ Instalação

1. Instale as dependências:

```bash
npm install
```

2. Execute o servidor de desenvolvimento:

```bash
npm run dev
```

3. Acesse o site em [http://localhost:3000](http://localhost:3000)

## 📦 Build para Produção

```bash
npm run build
npm start
```
